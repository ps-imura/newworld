AI運用指南書

この指南書は、株式会社 Poolside の新人が、Claude Code と Cursor を使って AI と協働しながら開発できるようになるためのガイドです。
ターミナルもプログラミングも初めて、という前提で書いています。順番に読んでも、必要な章だけ拾い読みしてもどちらでも構いません。

この指南書のゴール
最終章まで読み終えると、Claude Code と Cursor の基本操作ができ、AI と相談しながら簡単な Web ページ（HTML / CSS / JavaScript）を作って、実際にインターネット上に公開できる状態になります。

本書での約束ごと

「コマンド」とは
ターミナルに打ち込んで Mac に何かをさせるための文字列のこと。本書では $ ls のような形で書く（先頭の $ はプロンプト記号で、入力しなくてよい）。
「フォルダ」と「ディレクトリ」
同じものです。Mac の画面上では「フォルダ」、ターミナル上では「ディレクトリ」と呼ばれることが多い。本書では原則 「フォルダ」 に統一します。
「AI に頼んでみよう」と書いてある場所
Claude Code に貼り付けてみるとよい指示文（プロンプト）を載せてあります。コピペして実行してみて、返ってくる答えと自分の手元を見比べると理解が早くなります。
つまずいた時
第13章「Q&A」によくあるトラブルと対処法をまとめています。先にそこを開いてみてください。
